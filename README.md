# Twit-silici-rt-kaldirici
Twitterdaki gönderdiğiniz twitleri siler, rtleri geri çeker.

Google Chrome tarayıcısı yüklü olması lazım.

Dosyaları indirdikten sonra terminale "pip install -r requirements.txt" yazmanız yeterlidir. 
requirementsleri kurduktan sonra terminalden python3 delete.py komutunu girerek programı başlatabilirsiniz.
program başladıktan sonra tarayıcıdan twitterın login ekranına atacak, hesabınıza giriş yaptıktan sonra profilinize tıklayın ardından terminale girip enter'a basın.
tüm twit ve rtleri silmeye başlayacaktır.
